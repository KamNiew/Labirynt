# Labirynt
<h2>Gra na laboratoria z Grafiki i komunikacji człowiek-komputer</h2>
<p>Prosta gra Labirynt napisana w javie.</p><br/>
<p>Twoim celem jest dojście do drzwi zanim skończy się czas. By otworzć drzwi trzeba zebrać złote klucze. Srebrne klucze potrzeba tylko jednego, a kolejne powiększają punkty na konjec mapy. Gra zapętla mapy aż do przegranej (gdy skończy się czas)</p>
